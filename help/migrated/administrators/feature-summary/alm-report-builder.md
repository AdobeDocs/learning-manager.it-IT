# Report Builder in Adobe Learning Manager

## Panoramica

Crea, visualizza in anteprima e scarica report personalizzati con le colonne, i filtri e i dati scelti, senza dover ricorrere a strumenti di post-elaborazione.

Il Report Builder di Adobe Learning Manager offre agli amministratori un’area di lavoro di reporting autonoma per creare esattamente i report di cui hanno bisogno. Invece di scaricare un report fisso e rimodellarlo in uno strumento per fogli di calcolo, puoi selezionare le colonne desiderate, applicare filtri e scaricare un output pulito, tutto da un&#39;unica posizione.

## Perché è stato introdotto il Report Builder

I report esistenti in Adobe Learning Manager sono stati corretti. Ogni report include un elenco di colonne impostato che non è possibile modificare e le opzioni di filtro sono limitate. Gli amministratori che necessitavano di output personalizzati dovevano scaricare più report, unirli a Excel, applicare manualmente i filtri e ripetere il processo ogni settimana o mese.

Il Report Builder rimuove il ciclo. È possibile configurare un report una sola volta, salvarlo e scaricare nuovi dati ogni volta che è necessario, senza necessità di rimodellazione.

Tre problemi specifici risolti dal Report Builder:

* **Colonne fisse:** I report esistenti includono molte colonne che potrebbero non essere necessarie ed escludono quelle necessarie. Report Builder consente di selezionare solo i campi pertinenti per il tuo caso di utilizzo.
* **Filtro limitato:** Ora puoi filtrare in base a qualsiasi campo supportato. Ad esempio, includendo la data di iscrizione, la data di completamento, i campi attivi e le etichette del catalogo, non solo lo stato.

* **Origini dati incoerenti:** è possibile estrarre i report esistenti dall&#39;interfaccia utente, dall&#39;API dei processi o dai connettori, che potrebbero restituire dati leggermente diversi a seconda dei tempi. Il Report Builder si estrae da un&#39;unica origine dati coerente, in modo che ogni download rifletta gli stessi dati sottostanti.

## Adattamento del Report Builder ai report esistenti

Report Builder non sostituisce i report fissi esistenti in Adobe Learning Manager. Entrambi sono disponibili. Utilizzare i rapporti esistenti per gli output standard su cui si fa già affidamento. Usa Report Builder quando sono necessarie colonne personalizzate, filtri specifici o dati che al momento richiedono post-elaborazione.

## Dove trovare il Report Builder

Il Report Builder è disponibile nella home page dell&#39;amministratore. Seleziona **Report**, quindi seleziona **Report Builder**. Verranno visualizzate due schede: **Modelli** e **Report**.

![](assets/report-builder-0001.png)

## Chi può usare Report Builder

Il Report Builder è disponibile per gli amministratori.

>[!NOTE]
>
>Il Report Builder deve essere abilitato per gli account. Contatta il team dell&#39;account Adobe o il CSM se non è visibile nella vista dell&#39;amministratore.

## Procedure ottimali

* Se non conosci Report Builder, inizia con un modello. I modelli sono predefiniti per i casi d’uso più comuni e funzionano immediatamente.
* Salva ogni report configurato prima di scaricarlo. I report salvati possono essere scaricati nuovamente in un secondo momento senza riconfigurarli.
