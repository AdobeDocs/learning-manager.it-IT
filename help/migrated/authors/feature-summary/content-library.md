# Libreria dei contenuti

Scopri come creare contenuti da allineare ai corsi come contenuti autonomi.

## Libreria dei contenuti {#contentlibrary}

Il contenuto è la componente fondamentale di un corso. Gli Autori creano una libreria di contenuti che può essere allineata ai corsi come contenuti autonomi. Solo gli autori possono accedere a questa raccolta di contenuti.

## Aggiungere nuovo contenuto alla libreria {#addnewcontentinthelibrary}

Come accennato nella sezione precedente, puoi aggiungere contenuti interattivi e statici alla libreria.

## Aggiungere contenuto statico {#addstaticcontent}

1. Dopo aver effettuato l’accesso come Autore, fai clic su Libreria contenuti nel riquadro a sinistra, quindi su Aggiungi.

   In alternativa, puoi fare clic su Crea contenuto dalla pagina della Guida introduttiva.

1. Nel campo Nome, specificate un nome per il contenuto da caricare.
1. Nel campo Descrizione immettere la descrizione del contenuto. Accertarsi che la descrizione che si desidera immettere sia significativa. Il limite di caratteri è di 400 caratteri.
1. Per aggiungere il contenuto, fare clic su Aggiungi file di contenuto e caricare il file di risorse. Quando si aggiungono contenuti per più lingue, non è possibile combinare contenuti statici e interattivi in un unico gruppo. Tutti i contenuti in tutte le lingue devono essere statici oppure tutti i contenuti devono essere interattivi.

   Se si desidera sostituire il contenuto, è possibile sostituire un contenuto statico con un contenuto statico diverso. Lo stesso vale per i contenuti interattivi.

1. Nel campo Durata puoi facoltativamente immettere il tempo previsto che un Allievo trascorrerebbe in questo modulo. La durata è in minuti.

   Il tempo di apprendimento impiegato dall’Allievo viene calcolato in base alla durata specificata se l’Allievo ha contrassegnato un corso come completato. Se l’Allievo utilizza i contenuti nel lettore, il tempo impiegato nel lettore viene aggiunto al tempo di apprendimento impiegato. Se la durata effettiva del contenuto è inferiore alla durata specificata, non accade nulla in quanto il lettore rispetta sempre la durata effettiva per la visualizzazione.

1. Nel campo Tag contenuto, immetti i tag per il contenuto caricato in modo che il contenuto diventi rilevabile.

   L’Autore può utilizzare questi tag per cercare il contenuto durante l’aggiunta del contenuto al corso.

### Controllo delle versioni {#versioning}

La libreria dei contenuti mantiene anche il controllo delle versioni dei contenuti caricati. Se si apportano modifiche al contenuto, ad esempio una presentazione di PowerPoint, e si ricarica il file PPT nella libreria, il numero di versione viene incrementato di un&#39;unità. Ciò consente di tenere traccia delle modifiche apportate ai contenuti.

## Aggiungere contenuti interattivi {#addinteractivecontent}

1. Dopo aver effettuato l’accesso come Autore, fai clic su Libreria contenuti nel riquadro a sinistra, quindi su Aggiungi.

   In alternativa, puoi fare clic su Crea contenuto dalla pagina della Guida introduttiva.

1. Nel campo Nome, specificate un nome per il contenuto da caricare.
1. Nel campo Descrizione immettere la descrizione del contenuto. Accertarsi che la descrizione che si desidera immettere sia significativa. Il limite di caratteri è di 245 caratteri.
1. Per aggiungere il contenuto, fare clic su Aggiungi file di contenuto e caricare il file di risorse. Quando si aggiungono contenuti per più lingue, non è possibile combinare contenuti statici e interattivi in un unico gruppo. Tutti i contenuti in tutte le lingue devono essere statici oppure tutti i contenuti devono essere interattivi.

* [Tipi di file supportati](content-library.md#supported)*

  Il contenuto interattivo può essere un progetto SCORM, AICC o un progetto Captivate pubblicato. Il file deve essere un file zip.

  Puoi anche aggiungere contenuti HTML generati da Captivate, Presenter o Presenter Video Express.

[Tipi di file supportati](content-library.md#supported)

1. Learning Manager supporta i sottotitoli per i contenuti video caricati in Learning Manager. Ora gli Autori possono caricare il file contenente i sottotitoli insieme al file video.

   Gli Allievi possono quindi visualizzare i sottotitoli durante la riproduzione del modulo video.

   Il formato supportato è  [Tracce di testo video Web (webVTT)](https://www.w3.org/TR/webvtt1/).

   Il supporto per sottotitoli è disponibile per i contenuti video caricati nella Libreria dei contenuti di Learning Manager.

   In qualità di autore, quando caricherai un contenuto video o audio, puoi anche caricare il file .vtt che contiene i sottotitoli.

   I sottotitoli vengono quindi visualizzati nel lettore Fluidic. I sottotitoli sono inoltre conformi alle [Standard WCAG2.0](https://www.w3.org/TR/WCAG20/).

   Quando aggiungi un contenuto video alla libreria, puoi anche aggiungere il file vtt, che **deve** essere un file valido.

   ![](assets/webvtt.png)

   *Aggiungere un file webvtt*

   Il file VTT caricato corrisponde alla versione esistente del contenuto. In questo modo, il file caricato su webVTT non si collega alla versione precedente del contenuto.

   Nel caso in cui i contenuti siano stati creati in lingue diverse, potete caricare un file WebVTT diverso per ciascuna lingua. Gli Allievi potranno visualizzare i sottotitoli corrispondenti alla lingua selezionata durante la riproduzione.

   >[!NOTE]
   >
   >   Un file VTT supporta una sola lingua. Per supportare più lingue, carica più file video per ogni lingua di contenuto, quindi carica il rispettivo file VTT per ogni file video.

   Come Autore, ogni volta che modifichi il contenuto, video o audio, Learning Manager ti chiede un nuovo file vtt.

   Dopo aver aggiunto questo contenuto a un corso e quando lo visualizzi in anteprima come Allievo, puoi visualizzare i sottotitoli nel video.

   Sul lettore, attiva il pulsante CC sul lettore Fluidic per visualizzare o nascondere i sottotitoli.

   La stessa visualizzazione è presente nel **app per allievi** così come in **Anteprima come Allievo**.

   Quando **aggiungere, aggiornare o eliminare** nel file vtt, riceverai una notifica.
Il supporto del formato WebVTT non è disponibile per:

   1. Annunci video.
   1. Video riprodotto all’interno del contenuto di eLearning. Questo dipende dal contenuto.
   1. Video caricato in Apprendimento sociale.
   1. Video creato nell’app desktop Learning Manager.
   1. Contenuti video creati mediante il processo di migrazione.
   1. Riproduzione di video nell’app per dispositivi mobili in modalità offline.

1. Nel campo Durata puoi facoltativamente immettere il tempo previsto che un Allievo trascorrerebbe in questo modulo. La durata è in minuti.
1. Nel campo Tag contenuto, immetti i tag per il contenuto caricato in modo che il contenuto diventi rilevabile.

### Supporto per catalogo condiviso

Se un account del venditore condivide un catalogo che contiene i corsi e questi contengono i moduli, audio o video con i sottotitoli, i corsi devono comportarsi allo stesso modo nell’account dell’acquirente.

La propagazione del modulo deve funzionare correttamente dall&#39;account del venditore all&#39;account dell&#39;acquirente. Questo può includere: modifica/elimina/aggiunta del file vtt nel modulo.

Una volta caricati i contenuti, puoi visualizzare una notifica facendo clic sull’icona a forma di campana nell’angolo in alto a destra della pagina. Ogni volta che modifichi un contenuto e lo ricarichi, ricevi una notifica. Se apporti le modifiche, riceverai solo la notifica, non altri autori.

## Creare un quiz

Creazione di valutazioni all’interno di Adobe Learning Manager con il nuovo strumento per la creazione di quiz nella pagina Libreria dei contenuti. Le valutazioni create diventano parte della Libreria dei contenuti e possono essere aggiunte a una cartella &quot;pubblica&quot; per consentire il riutilizzo del corso.

1. Selezionate Libreria contenuti nel pannello a sinistra.
1. Nell’angolo in alto a destra dello schermo, seleziona **Aggiungi > Quiz**.
1. Nella pagina Crea quiz, digita il nome e la descrizione del quiz.
1. Nella sezione Contenuto quiz, seleziona **Aggiungi domanda per il quiz**.
1. Nella finestra di dialogo della domanda del quiz, seleziona il tipo di domanda. Sono disponibili tre tipi di domande:
   * Domanda a scelta multipla
   * True o false
   * Compila lo spazio vuoto
1. Immetti la domanda e seleziona la risposta corretta.
1. Impostate i punti per il quiz.
1. Se desideri che alla domanda venga fornita una risposta corretta per superare il quiz, seleziona la casella di controllo **Rispondere correttamente per superare il quiz**.
1. Seleziona **Salva e chiudi**.
1. Inserisci i punti per superare il quiz nel **Criteri di superamento** campo.
1. Se desideri che un Allievo visualizzi una risposta corretta, abilita l’interruttore **Mostra risposte corrette** agli allievi dopo il quiz.
1. Se desideri che le domande e le risposte vengano visualizzate in modo casuale, attiva gli interruttori:
   * Ordine domande casuale
   * Ordine delle opzioni di risposta casuale
1. Specificate una cartella in cui aggiungere il quiz per renderlo disponibile a tutti gli Autori.
1. Nella **Durata** , specifica il tempo che l’allievo deve dedicare al quiz.
1. Specificate un tag dall’elenco dei tag già creati.
1. Aggiungi un logo e uno sfondo al quiz.
1. Nell’angolo superiore destro della pagina, seleziona **Pubblica**.

Il quiz viene aggiunto alla Libreria dei contenuti. Come per qualsiasi contenuto nella Libreria dei contenuti, puoi ritirare un quiz e quindi eliminarlo.


## Aggiungi alla cartella {#add-folder}

Dopo che un Amministratore ha creato le cartelle dei contenuti, l’Autore può caricare un contenuto in una cartella dei contenuti, in modo che sia visibile solo a te o a un gruppo selezionato di Autori nell’account. Puoi anche rendere pubblico il contenuto e renderlo visibile a tutti gli autori dell’account.

**Esempio di utilizzo**

Ad esempio, le agenzie desiderano mantenere il pieno controllo del contenuto e chi lo sovrasta deve avere accesso a tutto il contenuto. Allo stesso tempo, i creatori di contenuti delle agenzie devono avere accesso solo ai propri contenuti e, in alcuni casi, ai contenuti di qualcun altro.

La libreria di contenuti con contenuto esistente (ad esempio, il contenuto caricato prima della configurazione delle cartelle dei contenuti) viene definita come **Cartella pubblica**. Impossibile ritirare o eliminare la cartella. Il contenuto che fa parte della cartella pubblica è accessibile a tutti i tipi di autori. Una volta configurate le cartelle dei contenuti, gli autori standard e gli autori personalizzati devono selezionare la cartella in cui deve essere inserito il contenuto durante il caricamento di nuovo contenuto.

>[!NOTE]
>
>Le cartelle pubbliche e private si escludono a vicenda. Questo significa che il contenuto **non può** essere contemporaneamente associato alla cartella pubblica e privata. Può essere associato alla cartella pubblica, **oppure** può essere associato a una o più cartelle private in qualsiasi momento.

Quando aggiungi un contenuto, puoi scegliere la cartella in cui risiederà il contenuto.

![](assets/add-to-content-folder.png)

*Aggiungi contenuto alla cartella*

Se si sceglie **Pubblico**, il contenuto sarà visibile a tutti gli autori. Per impostazione predefinita, tutti i contenuti presenti nell’account che non fanno parte di alcuna cartella vengono inseriti nella cartella pubblica.

Tieni presente che le cartelle dei contenuti sono semplicemente scomparti virtuali per collegare il contenuto. Nel caso in cui un contenuto venga inserito in due cartelle, significa che il file di contenuto è sempre un singolo file ma collegato a più cartelle. Pertanto, nel caso in cui il contenuto venga aggiornato dall&#39;autore personalizzato-1 che ha accesso a custom-folder-1, lo stesso contenuto aggiornato si rifletterà anche in custom-folder-2 a cui accede l&#39;autore-2 personalizzato.

Nella Libreria dei contenuti sono disponibili due opzioni per la gestione delle cartelle dei contenuti:

**Tutte le cartelle**

Si tratta di un elenco che visualizza tutte le cartelle create nell’account.

![](assets/list-of-all-folders.png)

*Visualizza tutte le cartelle*

**Tutti gli autori**

È un elenco che mostra gli autori che hanno creato contenuti e li hanno caricati nella libreria.

![](assets/list-of-all-authors.png)

*Visualizza tutti gli autori*

Questo è disponibile **solo** quando un Amministratore crea una nuova cartella.

## Sposta il contenuto nella cartella {#movecontenttofolder}

Per spostare il contenuto di una cartella pubblica in una cartella privata,

1. Seleziona **Pubblico** dalla cartella **Tutte le cartelle** elenco a discesa.

   ![](assets/list-of-public-folders.png)

   *Visualizza tutti i contenuti caricati*

   ![](assets/move-content-to-folder.png)

   *Sposta nella cartella il contenuto selezionato*

1. Scegli la cartella in cui spostare il contenuto. Fai clic **[!UICONTROL Sposta]**.

Dopo aver aggiunto il contenuto, puoi modificare i criteri di completamento per il contenuto.

In Learning Manager, i distintivi e le abilità vengono assegnati in base al successo e al completamento. Se l’Allievo ha completato un corso ma non riesce, non riceve il badge e l’abilità corrispondenti all’LO.

Ad esempio, se hai utilizzato Adobe Captivate per creare il tuo corso e hai impostato i parametri di apprendimento nella finestra di dialogo Preferenze, le stesse impostazioni vengono migrate su Learning Manager nelle opzioni Criteri di completamento.

Nella sezione Criteri di completamento è possibile impostare le opzioni riportate di seguito:

**All’Avvio Del Contenuto:** Se attivi questa opzione, definisci i criteri di completamento per il contenuto quando un Allievo apre il contenuto.

**In base alla percentuale minima richiesta:** Imposta un valore come percentuale minima di utilizzo da parte dell’Allievo. Ad esempio, se imposti la percentuale su 50, l’Allievo può utilizzare il 50% del contenuto e soddisfare comunque i criteri di completamento.

**Quiz:** Scegliere uno dei seguenti criteri:

* **Quiz superato:** Lo stato viene segnalato come Completato solo se un Allievo supera il quiz.
* **Tentativo di quiz:** Lo stato viene segnalato come Completato se gli Allievi tentano di rispondere al quiz indipendentemente dal fatto che il quiz venga superato o meno.
* **Quiz superato o limite raggiunto:** Lo stato viene segnalato come Completato se gli Allievi superano il quiz o hanno effettuato tutti i tentativi. Ad esempio, se il numero di tentativi impostati nel corso è due, e:

   * Se gli Allievi accettano il primo tentativo e lo superano, lo stato viene segnalato come Completato e Superato.
   * Se gli Allievi effettuano il primo tentativo e falliscono, lo stato viene segnalato come Incompleto e Non riuscito poiché il limite dei tentativi non è ancora stato raggiunto.
   * Se gli Allievi rispondono nuovamente al quiz ma non riescono, lo stato viene segnalato come Completato e Non riuscito.
   * Se gli Allievi tentano di ripetere il quiz e lo superano, lo stato viene segnalato come Completato e Superato.

## Imposta criteri di successo {#setsuccesscriteria}

Allo stesso modo, puoi definire i criteri di successo per il corso. Un criterio di successo indica che le prestazioni di un Allievo sono Superate o Non riuscite. Se hai creato un corso in Captivate, puoi impostare i criteri di successo per il corso nella finestra di dialogo Preferenze, come illustrato di seguito:

Ad esempio, è stato caricato un modulo contenente un quiz. Ora hai impostato i criteri di completamento per quel modulo su All’avvio del contenuto e i criteri di successo su Quiz superato.

Se l’Allievo ha avviato il corso e ha fallito il quiz, il corso verrà contrassegnato come Completato, tuttavia i criteri di successo verranno soddisfatti solo quando l’Allievo supera il quiz.

## Ritiro del contenuto {#retirecontent}

Una volta pubblicato un contenuto, non è possibile eliminarlo. Prima devi ritirare il contenuto. Quando contrassegni un contenuto come Ritirato, il contenuto non è più visibile agli Allievi. Anche il contenuto viene spostato nella sezione Ritirato. Puoi anche spostare il contenuto nello stato pubblicato in un secondo momento.

Per ritirare il contenuto, effettua le seguenti operazioni:

* Nella raccolta Contenuti selezionare il contenuto che si desidera ritirare.
* Seleziona Azione > Ritira.

I contenuti utilizzati in qualsiasi oggetto di apprendimento non subiscono modifiche. Gli Allievi possono comunque continuare ad accedere al contenuto.

## Ripubblicare i contenuti ritirati {#republishretiredcontent}

Una volta ritirato un contenuto, puoi ripubblicare il contenuto e farlo apparire nell&#39;elenco Pubblicato. Ad esempio, se si è ritirata la versione 1 di un contenuto e si desidera sostituirla con la versione 2, è possibile spostare la versione 1.pptx, ad esempio, nell&#39;elenco Pubblicato e aggiornare il file con la versione 2.pptx. Il nuovo file diventa disponibile per l’utilizzo in vari corsi.

Per ripubblicare il contenuto ritirato:

1. Passare alla **Ritirato** e selezionare il contenuto da ripubblicare.
1. Seleziona **Azione** > **Ripubblica**.

Il contenuto viene visualizzato nell&#39;elenco Pubblicato.

## Elimina contenuto {#deletecontent}

Dopo aver ritirato un contenuto, puoi eliminarlo.

* Passa alla scheda Ritirato e seleziona il contenuto da eliminare.
* Selezionate Azione > Elimina.

I corsi esistenti che utilizzano il contenuto, che vengono eliminati dalla libreria dei contenuti, continueranno a utilizzare il contenuto.

## Domande frequenti {#frequentlyaskedquestions}

+++ Come si carica un contenuto SCORM in Adobe Learning Manager?

Creazione di un corso di e-learning conforme allo standard SCORM in qualsiasi strumento, ad esempio Adobe Captivate, e pubblicazione del contenuto come file zip. Quindi in Learning Manager, carica il file zip nel catalogo e imposta i criteri di completamento e successo.
+++

+++Come si carica una nuova versione dello stesso contenuto in Learning Manager?

In Learning Manager, la libreria dei contenuti conserva anche le versioni dei contenuti caricati. Se si apporta una modifica al contenuto, ad esempio una presentazione di PowerPoint, e si ricarica la presentazione nella libreria, il numero di versione viene incrementato di una unità. Ciò consente di tenere traccia delle modifiche apportate ai contenuti. Una nuova versione del contenuto può essere applicata contemporaneamente a tutti gli Oggetti di apprendimento oppure puoi applicare singoli aggiornamenti per ogni corso.
+++

+++Come si modificano i dettagli di un corso in una lingua diversa?
Dopo aver aggiunto una o più lingue, come descritto nella sezione precedente, fai clic sulla scheda di ciascuna lingua e aggiungi/modifica le informazioni sul corso.
